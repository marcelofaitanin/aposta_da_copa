# 2. Layout das Telas (Wireframes)

Wireframes de baixa fidelidade das principais telas do sistema, em formato SVG
(abrem direto no navegador e renderizam no GitHub).

| Tela                | Arquivo                                                  | Caso de uso |
| ------------------- | ------------------------------------------------------- | ----------- |
| Login               | [`login.svg`](login.svg)                                | UC02        |
| Jogos + Aposta      | [`jogos.svg`](jogos.svg)                                | UC03, UC04  |
| Minhas Apostas      | [`minhas-apostas.svg`](minhas-apostas.svg)              | UC05        |
| Painel Admin        | [`admin.svg`](admin.svg)                                | UC07        |
| Diagrama de Casos de Uso | [`diagrama-casos-de-uso.svg`](diagrama-casos-de-uso.svg) | —      |

> A tela de **Cadastro** segue o mesmo layout da tela de Login, com um campo
> adicional de "Nome".

Os wireframes representam a estrutura e o posicionamento dos elementos. O
**layout final implementado** (com cores e tipografia) pode ser visto rodando o
projeto (`ng serve`) ou nos mockups HTML em [`../layout-html/`](../layout-html/).
