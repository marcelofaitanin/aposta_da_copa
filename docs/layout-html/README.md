# 3. Layout HTML

Mockups **estáticos em HTML/CSS puro** (sem Angular) das telas principais,
reproduzindo o layout implementado na aplicação. Basta abrir os arquivos `.html`
no navegador (duplo clique).

| Tela            | Arquivo                    | Componente Angular equivalente              |
| --------------- | -------------------------- | ------------------------------------------- |
| Tela de Jogos   | [`jogos.html`](jogos.html) | `frontend/src/app/pages/games/games.html`   |
| Tela de Login   | [`login.html`](login.html) | `frontend/src/app/pages/login/login.html`   |

> O layout HTML "vivo" (dinâmico, consumindo a API) está nos **templates dos
> componentes Angular** em `frontend/src/app/pages/*/*.html`. Os arquivos desta
> pasta são versões estáticas equivalentes, para visualização rápida do layout
> sem precisar subir o projeto.
